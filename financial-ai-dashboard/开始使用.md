## 🔧 手动启动（可选）

### 启动后端
```bash
cd backend
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install websockets
python server.py
```

### 启动前端
```bash
cd frontend
npm install
npm start
```
