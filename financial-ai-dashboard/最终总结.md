## 🚀 快速启动

### 方式一：自动启动（推荐）

**macOS / Linux:**
```bash
cd financial-ai-dashboard
chmod +x start.sh
./start.sh
```

**Windows:**
双击 `start.bat` 文件

### 方式二：手动启动

**后端（终端 1）:**
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python server.py
```

**前端（终端 2）:**
```bash
cd frontend
npm install
npm start
```

### 访问地址

```
🌐 前端界面：http://localhost:3000
🔌 WebSocket: ws://localhost:8080
```
