## 🔍 问题诊断步骤

### 方法一：浏览器控制台检查（推荐）

1. **打开浏览器开发者工具**
   - 按 `F12` 或右键点击页面选择"检查"
   
2. **查看 Console（控制台）**
   ```
   查找以下关键信息：
   - ✅ "🚀 初始化金融 AI 组件..." - 表示 JS 正常执行
   - ⚠️ 任何红色错误信息
   - ⚠️ iframe 加载失败的警告
   ```

3. **查看 Network（网络）**
   ```
   刷新页面，检查以下资源是否成功加载：
   - /css/widget-components.css (状态码应该是 200)
   - /js/widget-tabs.js (状态码应该是 200)
   - /components/stock-chart-widget.html (状态码应该是 200)
   - /components/crypto-dashboard-widget.html (状态码应该是 200)
   ```

4. **检查 Elements（元素）**
   ```
   在 DOM 中查找：
   - .widget-section 元素是否存在
   - .tab-content 是否设置了正确的 display 属性
   - iframe 的 src 属性是否正确
   ```

### 方法四：检查 iframe 高度

有时候 iframe 可能因为高度为 0 而看不见。

**检查方法：**
1. 打开开发者工具（F12）
2. 点击元素选择器（左上角的小箭头图标）
3. 点击 widget-section 区域
4. 查看右侧 Styles 面板中的 `.widget-iframe` 样式
5. 确认 `height` 属性是否为 `650px`

**临时修复（如果发现高度问题）：**
在控制台执行：
```javascript
document.querySelectorAll('.widget-iframe').forEach(iframe => {
  iframe.style.height = '650px';
  iframe.style.display = 'block';
});
```

## 🛠️ 常见问题及解决方案

### 问题 1：组件显示空白

**可能原因：**
- JavaScript 执行错误
- Vue/React 库加载失败

**解决方案：**
```bash
# 重新生成并部署
hexo clean
hexo generate
hexo deploy
```

---

### 问题 2：Tab 切换无反应

**可能原因：**
- `widget-tabs.js` 未执行
- 事件监听器未绑定

**调试方法：**
打开控制台，手动执行：
```javascript
// 检查函数是否存在
console.log(typeof window.switchWidgetTab);
// 应该输出 "function"

// 手动切换到股票预测
switchWidgetTab('stock');

// 手动切换到加密货币
switchWidgetTab('crypto');
```

---

### 问题 3：样式错乱

**可能原因：**
- CSS 文件未加载
- 样式冲突

**检查方法：**
在控制台执行：
```javascript
// 检查样式表是否加载
const styles = document.querySelectorAll('link[rel="stylesheet"]');
styles.forEach(link => console.log(link.href));

// 查找 widget-components.css
const widgetStyles = [...styles].filter(s => s.href.includes('widget-components'));
console.log('Widget CSS 已加载:', widgetStyles.length > 0);
```

---

### 问题 4：移动端不显示

**可能原因：**
- 响应式样式问题
- 媒体查询覆盖

**临时测试：**
在桌面浏览器按 `F12`，切换到移动设备模式查看。

## 🔄 自动化检测脚本

创建一个简单的检测脚本，在浏览器控制台运行：

```javascript
(function() {
  console.log('🔍 开始检测金融 AI 组件...');
  
  // 检查 HTML 结构
  const widgetSection = document.querySelector('.widget-section');
  console.log(widgetSection ? '✅ Widget Section 存在' : '❌ Widget Section 不存在');
  
  // 检查 Tab 按钮
  const tabButtons = document.querySelectorAll('.tab-btn');
  console.log(tabButtons.length > 0 ? `✅ 找到 ${tabButtons.length} 个 Tab 按钮` : '❌ Tab 按钮不存在');
  
  // 检查 iframe
  const iframes = document.querySelectorAll('.widget-iframe');
  console.log(iframes.length > 0 ? `✅ 找到 ${iframes.length} 个 iframe` : '❌ iframe 不存在');
  
  // 检查 JS 功能
  console.log(typeof window.switchWidgetTab === 'function' ? '✅ Tab 切换功能正常' : '❌ Tab 切换功能异常');
  
  // 检查 CSS
  const widgetContainer = document.querySelector('.widget-container');
  const bgStyle = window.getComputedStyle(widgetContainer).background;
  console.log(bgStyle && bgStyle !== 'none' ? '✅ 样式已应用' : '❌ 样式未应用');
  
  console.log('🔍 检测完成');
})();
```

将此代码粘贴到浏览器控制台（F12 -> Console），按回车即可查看检测结果。
