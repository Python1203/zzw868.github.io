CSS Fade-In Animation

演示简单的CSS淡入动画效果

动画原理

这个淡入动画使用了CSS的@keyframes规则，从透明度0开始，逐渐过渡到完全不透明。同时还添加了轻微的向上移动效果，使动画更加自然。

动画时长为2秒，使用ease-in-out缓动函数，创建平滑的过渡效果。

使用方法

只需为元素添加fade-in类名即可应用动画效果。可以通过animation-delay属性控制多个元素的动画顺序。

这种动画技术在现代网页设计中非常常用，用于提升用户体验和页面视觉效果。

CSS动画代码

/* 淡入动画类 */ .fade-in { opacity: 0; animation: fadeIn 2s ease-in-out forwards; } /* 关键帧动画 */ @keyframes fadeIn { from { opacity: 0; transform: translateY(20px); } to { opacity: 1; transform: translateY(0); } }
