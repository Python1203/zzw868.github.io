## 🏃 快速开始

### 方式一：访问组件导航中心 (推荐)

直接在浏览器中打开:
```
/components-hub.html
```

这个页面提供了所有组件的美观导航界面，包含:
- ✨ 精美的卡片设计
- 🔍 组件描述和技术标签
- 🎨 统一的视觉风格
- 📱 完全响应式设计

### 方式二：访问综合演示页面

直接在浏览器中打开:
```
/all-components-demo.html
```

这个页面使用 iframe 嵌入了所有组件，可以一站式查看所有效果。

### 方式三：直接访问单个组件

在浏览器中直接打开对应的 HTML 文件，例如:
```
/test-stock-chart.html
/crypto-dashboard.html
/contact-form.html
```

### 方式四：通过主页访问

打开 `index.html`,在特色功能展示区点击相应按钮即可访问核心组件。

## 🚀 部署说明

### 本地测试

1. 直接在浏览器中打开任意 HTML 文件
2. 或使用本地服务器:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server
```

### GitHub Pages 部署

1. 确保所有文件在项目根目录
2. 推送到 GitHub 仓库
3. 启用 GitHub Pages 功能
4. 访问: `https://yourusername.github.io/components-hub.html`

### 自动部署脚本

使用项目中的自动部署脚本:
```bash
./auto-deploy.sh
```

## 💡 使用技巧

### 1. 组件组合使用

你可以将多个组件组合在一起创建更丰富的页面:

**示例**: 
- 导航栏 + Grid 布局 + 卡片组件 = 产品展示页
- 联系表单 + 淡入动画 = 优雅的联系我们页面
- 按钮 + 模态弹窗 = 交互式操作界面

### 2. 自定义样式

所有组件都使用了 CSS 变量，方便统一修改:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    /* 修改这些变量来改变主题色 */
}
```

### 3. 响应式优化

每个组件都已针对移动设备优化，可以直接在手机和平板上使用。

### 4. 性能优化建议

- 按需加载组件，不要一次性加载所有组件
- 使用 CDN 加速外部资源 (React, Vue 等)
- 压缩 CSS 和 JavaScript 文件

### 5. SEO 优化

- 为每个页面添加合适的 meta 标签
- 使用语义化的 HTML 结构
- 添加适当的 alt 文本和 ARIA 标签

本项目遵循以下设计原则:
